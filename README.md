# c-vim

一直特别喜欢 k-vim 的配置，但随着自己技术栈变得越来越有针对性。

所以有必要针对 k-vim 进行重新的定制，来满足我对日常开发的需求，这就是 c-vim 诞生的原因。

特别感谢 k-vim

## 安装

```
brew install macvim --with-luajit --with-lua --with-python3
git clone https://github.com/twocucao/c-vim.git your/path
cd your/path
sh -x install.sh
```
