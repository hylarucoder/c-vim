# c-vim

按个人需求对 vim-init 做少许改动

## Feature

### Code Completion

### Language

1. Python
2. Rust
3. Golang
4. Markdown

## 安装

```
brew install neovim --head
mkdir -p ~/Cystem && cd ~/Cystem
git clone https://github.com/twocucao/c-vim.git
ln -sfv `pwd`  ~/.config/nvim
```
